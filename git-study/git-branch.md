# git branch

- ##### 显示所有的branch
  ``` git branch -a ```
  ```
  eg:
  Jose@Jose-PC MINGW64 /e/git-repository/first-pr-4-test/first-pr (gh-pages)
  $ git branch -a
  * gh-pages
    remotes/origin/HEAD -> origin/gh-pages
    remotes/origin/feature/move-jquery-from-cdn-to-local
    remotes/origin/gh-pages
```
