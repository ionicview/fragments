```
Ctrl  
  Ctrl  + Delete （消除掉右边的单词）
  Ctrl  + Backspace （消除掉左边的单词）
  Ctrl  + W（选择现在位置上的单词）
  Ctrl  + D （剪贴所选单词）
  Ctrl  + U （剪贴到行头 ）
  Ctrl  + K （剪贴到末尾）
  Ctrl  + E （剪贴所选行）
  Ctrl  + H (将光标挪到页面中间）
  Ctrl + J （将光标移动到指定行）
  Ctrl + [ （对括号的检测）·
Shift
  Shift +F10 （行的重复）
  Ctrl  + Shift （削除）
  Ctrl + Shift + D   （删除选中的单词）
  Ctrl + Shift + U   （删除到开头）
  Ctrl + Shift + K　（删除到末尾）
  Ctrl + Shift + E    （删除整行）
  Ctrl + Shift + E （添加各种补充）（其实是搜狗的快捷键）
  Ctrl + Shift + ← （选择单词的左边）
  Ctrl + Shift + → （选择单词的右边）
Alt
  Alt  + “+” 插入日期
  Alt  + “;” 插入时间
其他
  F3 下一个的检索
  F4 上下 分割
  shift +F3 上一个检索
  Ctrl + Shift + Alt  + →（windows纵向最宽）
  Ctrl + Shift + Alt  + ↑（windows纵向最宽）
  Ctrl  + Alt
  Ctrl  + Alt  ->等，矩形选择，和Alt光标一样
```
